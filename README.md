Jedná se o konzolovou hru v Pythonu inspirovanou kostkovými hrami z RPG prostředí (např. Kingdom Come: Deliverance). Projekt je navržen objektově (OOP) a klade důraz na čistou architekturu, rozšiřitelnost a práci se stavem hráčů.¨

Hlavní vlastnosti projektu:
-Objektově orientovaný návrh (Player, AIPlayer, Game, Dice, RollSet)
-Více herních režimů s různými cílovými skóre a výší sázek
-Plně funkční bodovací systém včetně speciálních kombinací kostek
-Persistentní hráči – ukládání jména a herní měny (coins) do JSON databáze
-Automatické načítání hráče při nové hře

Technologie: 
-Python 3
-OOP design
-Práce se soubory (JSON)
-Základy herní logiky a stavových automatů
-Jednoduchá AI logika (risk management)
